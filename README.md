### Primer paso:
Crear un entorno virutal:
```

```

### Segundo paso:
Instalar librerias necesarias:
```
pip install -r requirements.txt
```
### Tercer paso:
Inicializar el proyecto:
```
python manage.py runserver
```

Para ejecutar los tests de la API ejecutar:
